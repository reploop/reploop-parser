今天的嘉宾演讲主题比较分散，所以，我们一个一个快速的过，除了Tim Soulo分享Ahrefs的一些高级用法没有PPT外，其他的都在Google Drive里，发送邮件给我索要。

1. Gael Breton: The “Kaizen” Approach to Site Building

Gael Breton是Authority Hacker的联合创始人。

看到标题里的引号文字了吗？估计你应该跟我一样，不知道什么是Kaizen吧。

谷歌一下，就知道了。

没错，就是中文的音译。

如果你了解Gael的话，会发现他对我们中国文化还是略有知晓的，这源于他六年前在HK读过书，并去过我们内地，上面的采访视频里有提过。

Anyway，回归主题。

Gael的演讲围绕着怎么通过20%的努力获得80%的效果这个思维来优化你的内容站（content sites），让你的目标关键词排的更靠前、获取更多优质SEO流量。

开头，他讲到了今年年初的两个小收获：

一是：更改一篇博客文章的update date后排名上涨，原来Google freshness真的有用；

二是：使用Website Auditor的TF*IDF进行on-page优化后，排名有上涨，虽然不是全部。

这些小收获让他去思考怎么通过最小的努力获取最大的回报，于是他将这一原则应用到了英文内容站建设的各个方面，并得出一些实用的结论：

  • 别太过分关注搜索量（keyword volumn），还得看看SERP的点击率和前几名的流量（通过Ahrefs的click data）。比如：有些词搜索量巨大，但几乎不产生点击行为，那你就算拍上去了也是白搭，不会有多少流量。
  • Build hubs, not pages。也就是silo的意思，把内容和主题相近的那些文章归类在一个hub里。
  • 内链的效果不比外链效果差，而且内链锚文本随意，关键词锚文本也可以。
  • 给谷歌想要的内容，而不是你认为最高质量的内容，什么是谷歌想要的内容呢，用Surfer SEO去发现SERP前几名的一些共性，然后整理出一份谷歌喜欢的内容框架，让写手去创作。
  • 精选摘要（featured snippets），去学习Hubspot吧，他们网站开发了专门给谷歌看的HTML代码，很干净！
  • 使用Google Search Console的URL Inspection去“实时”测试（24小时内，同一个页面，可以提交多次哦，直至谷歌让你明天再来尝试）。
  • SERP里有一半的Title都是被谷歌修改过的，越短的title（50字符左右）表现越好。
  • 发送外链获取emails时，casual tone、精简、多次提及对方的名字等都有助于提升转化率。
  • 周一和周五发送邮件，回复率和转化率略比其他时间高。
  • DR 30-50的网站更容易获得外链。
  • 一个页面的affiliate links多少与该页面的排名表现没有相关性（意思是：commercial pages不用在意有太多的affiliate links）。

另外，Authority Hacker Pro在未来36小时内开放购买（你现在看到时，还有24小时左右），如果有需要或感兴趣，请联系我（seoactionblog@gmail.com）。

2. Travis Jamison: Think Like an Investor with SEO

Travis是Smash Digital (前称Supreme SEO)的创始人，本次演讲他主要谈的是怎么通过投资思维去增长一个SEO项目，并分享了过去几年利用这一思维，他成功变现了7个规模不一的项目，项目金额从五位数（几万）到8位数（几千万）美元不等。

他首先讲了与天使投资（angel investment）的一些概念和基本原则。

比如：投资者一般喜欢投有增长空间的startup，这样才有机会获得资本回报（capital gain），尽管有很高的risk。

SEO项目跟这有啥关系呢？有！

比如：当一个投资者看到这么一个网站 bestrobotvacuumcleanerhq.net 你觉得他会投资吗？你觉得这个站会增长到7到8位数吗？

不可能！

所以，think big and create a brand!

如果你是一个review站，当你获得一定流量后，尝试着sell your own products，创建一个community, 如果可以的话，去开发一个SaaS产品卖给你的受众。

价值投资的另一个原则便是：用75美分的成本换回1美元的回报！

Travis曾以19倍（低价）买过一个内容站，后来增长得不错。

总的感觉，Travis的这个演讲非常有启发，不拘泥于SEO的细节，从投资的角度去看待一个SEO项目或网站，视野很开阔。

如果你搞英文内容站的目的是出售变现，强烈建议你看看Travis的slides。

3. Tim Soulo: Nine Things That Only Ahrefs Can Do For You

Tim介绍了Ahrefs的九个独有功能，让我们SEO的内容和外链研究更加高效。

比如：click data, top-folder, keyword insurance, keyword alert, 用Content Explorer去做link prospecting等。

因为大会屏幕分辨率不是太高，Tim Soulo的演讲几乎都是Ahrefs工具的截图，所以，就不一一在这里介绍了。

回头问问他能否把PPT给我，感兴趣的邮件啊。

4. Nick Nimman: YouTube Mastery

Nick是知名的YouTuber（YouTube红人），其油管频道拥有近五十万粉丝。

本次演讲，他主要与我们分享了YouTube SEO的一些基本和中高级技巧。

因为我本人没有做过YouTube视频营销，所以就只能乖乖的做笔记了。

Nick首先分享了YouTube的基本用户数据，比如：它是除了谷歌以外的第二大搜索引擎，地球上48%的人都在用它。

YouTube视频流量来源主要包括：

  • YouTube内部搜索
  • Suggested Videos（YT推荐视频）
  • Google.com搜索
  • 第三方网站嵌入

接着，他分享了YouTube的排序规则和优化技巧：

  • Keyword in title, and description in a natural way
  • Click through rate (most important)
  • Thumbnail
  • Impressions that lead to watch time
  • Watch time (per video basis)
  • Audience attention
  • Number of comments
  • Pinned comment
  • User rating (thumb up/down)
  • Hashtag
  • Authority (topic relevance)

应该没有记录完全，做油管视频营销想要了解更多的伙伴们，可以看看他的slides学习。

5. Grant Simmons: Rock Solid SEO

Grant是Homes.com负责搜索营销的副总裁（VP）。

他的职业经历全是围绕着优化大型网站（big site）的SEO表现，比如：ups, fox sports, nissan, puma等知名品牌的官网。

Grant的演讲非常幽默，但很多笑点我也get不到，估计只有在欧美生活的人才可能觉得好笑。

本次他的演讲围绕着大站的品牌策略、网站架构、内链、内容、外链等，我觉得讲得比较浅，更多的偏向概念普及层面。

这里也不一一分享了，维护大站的SEO伙伴们自行前往他的PPT去学习吧。
